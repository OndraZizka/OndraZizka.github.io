JBoss Operation Network 2.3.0: Tomcat not supported on Windows 2003 and 2008

For the record: You won't get Tomcat to work on JON 2.3.0. It has problems with process scanning, using Sigar. See the log bellow. Note the part with Unexpected error occurred while looking up info for [<unknown>] process with pid [288] - call to getProcExe failed. Did the process die? Cause: org.hyperic.sigar.SigarException: Incorrect function.
